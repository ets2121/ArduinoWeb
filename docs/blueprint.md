# **App Name**: Arduino IDE Mirror

## Core Features:

- Code Editor: A multi-tabbed text editor for writing and editing Arduino code.
- Code Compilation: Compile Arduino code directly within the application to check for errors.
- Board and Port Selection: Select the target Arduino board and communication port.
- Code Upload: Upload compiled code to the selected Arduino board.
- Serial Monitor: Display serial communication between the Arduino board and the application.
- Library Management: Install and manage Arduino libraries within the application.
- Examples: Incorporate an easy access to several arduino projects to inspire the user and give him some start-up projects.

## Style Guidelines:

- Primary color: Dark gray (#343A40) for the main background, replicating the dark theme.
- Background color: Very dark gray (#212529) to provide a subtle contrast against the primary background.
- Accent color: Yellow (#FFC107) to highlight active elements and important notifications.
- Body and headline font: 'Inter' (sans-serif) for clear readability in both code and UI text.
- Code font: 'Source Code Pro' (monospace) for displaying code with consistent character spacing.
- Use simple, monochrome icons for actions and features to maintain a clean interface.
- Follow a multi-pane layout, with code editor, serial monitor, and board settings accessible through tabs or panels.
- Subtle transition animations when switching tabs or displaying notifications.
